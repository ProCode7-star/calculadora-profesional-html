# Calculadora Profesional HTML

Una calculadora web moderna y profesional desarrollada con HTML, CSS y JavaScript. Interfaz intuitiva y responsive para realizar operaciones matemáticas de forma rápida y eficiente.

## 🎯 Características

- ✅ Operaciones matemáticas básicas (suma, resta, multiplicación, división)
- ✅ Interfaz moderna y profesional
- ✅ Diseño responsive (adaptable a cualquier dispositivo)
- ✅ Historial de cálculos
- ✅ Botones de acceso rápido (Limpiar, Borrar)
- ✅ Soporte para decimales
- ✅ Fácil de usar

## 🚀 Inicio Rápido

### Prerrequisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación adicional

### Uso
1. Descarga o clona el repositorio
2. Abre el archivo `index.html` en tu navegador
3. ¡Comienza a realizar cálculos!

```bash
git clone https://github.com/ProCode7-star/calculadora-profesional-html.git
cd calculadora-profesional-html
```

## 📁 Estructura del Proyecto

```
calculadora-profesional-html/
├── index.html          # Página principal
├── style.css           # Estilos profesionales
├── script.js           # Lógica de la calculadora
└── README.md          # Este archivo
```

## 💻 Tecnologías Utilizadas

- **HTML5** - Estructura y semántica
- **CSS3** - Estilos y diseño responsive
- **JavaScript (Vanilla)** - Lógica de cálculos

## 🎨 Diseño

La calculadora cuenta con:
- Interfaz limpia y minimalista
- Colores profesionales
- Botones intuitivos
- Pantalla clara para mostrar resultados
- Tipografía legible

## 📝 Ejemplo de Uso

1. Ingresa el primer número
2. Selecciona la operación deseada (+, -, ×, ÷)
3. Ingresa el segundo número
4. Presiona "=" para obtener el resultado
